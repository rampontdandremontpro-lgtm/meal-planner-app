    • findMealById(id: string): Promise<ExternalRecipe | null>

      Récupère le détail d'une recette externe par son identifiant.

      Cette méthode interroge l'API TheMealDB avec lookup.php?i=... puis harmonise la réponse obtenue.

      Règles métier :

      • si aucun résultat n'est trouvé, la méthode renvoie null ;
      • la recette retournée suit le format externe attendu par l'application.

      Parameters

      • id: string

        Identifiant externe de la recette dans TheMealDB.

      Returns Promise<ExternalRecipe | null>

    searchMeals

    • searchMeals(search?: string): Promise<ExternalRecipe[]>

      Recherche des recettes externes via TheMealDB.

      Cette méthode interroge l'API TheMealDB pour récupérer une liste de recettes publiques. Si aucun mot-clé n'est fourni, un fallback sur la lettre a est utilisé afin de renvoyer des résultats par défaut.

      Règles métier :

      • si search est renseigné, la recherche se fait avec search.php?s=... ;
      • sinon, un fallback utilise search.php?f=a ;
      • les recettes renvoyées sont harmonisées dans le format attendu par le backend.

      Parameters

      • Optionalsearch: string

        Mot-clé de recherche optionnel.

      Returns Promise<ExternalRecipe[]>

      Une liste de recettes externes harmonisées.
