# ADR-008 — Utilisation de Swagger

## Statut
Accepté

## Contexte
Le projet nécessite une documentation claire des routes API.

## Décision
Utilisation de Swagger avec NestJS.

## Conséquences positives
- Documentation automatique
- Tests faciles des routes
- Visualisation claire

## Conséquences négatives
- Ajout de décorateurs

## Alternatives envisagées
- Documentation manuelle
